JSON ➔ Up?

It is a simple website monitoring tool ( I think this is a nice easy project to do and works on golang's strengths ). example json from monitored server:

  [{name: 'server1.freedisk', value: 200, status: 'OK'}, {name...}].

It will be simple, and do two things: alert if a json status is bad, and draw a sparkline for that status. The sparkline graph will be drawn using either a metric in the json or the response time. I registered the domain jsonup.com. channel on slack #jsonup
